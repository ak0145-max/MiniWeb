Hello! This is my **Chattogram Wonders** image gallery project. I created this as a beginner HTML and CSS assignment to showcase the natural beauty of Chattogram Division in Bangladesh.

Chattogram Division is blessed with some of the most stunning landscapes in the country — from the world’s longest sea beach to misty hills and peaceful lakes. In this project, I have highlighted six beautiful places that truly represent the charm and diversity of this region.

### Featured Places:
- **Cox’s Bazar** – The world’s longest natural sea beach  
- **Bandarban** – The Queen of Hills  
- **Khagrachari** – Green hills and rich indigenous culture  
- **Rangamati** – The Lake District of Bangladesh  
- **Potenga Beach** – A popular spot near Chattogram port  
- **Kaptai Lake** – The largest man-made lake in Bangladesh  

---

### ✨ What I Built

I designed a clean, modern, and fully responsive image gallery. When you hover over any image, it gently zooms in with a nice smooth effect. The layout looks good on mobile phones, tablets, and desktops. I paid attention to colors, spacing, and typography to make the page visually appealing.

This project helped me practice HTML structure, CSS Grid, Flexbox, hover animations, and responsive design.

### 🛠 Technologies Used
- HTML5  
- CSS3 (Grid, Flexbox, Transitions)  
- Responsive Web Design
